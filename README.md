## 개발 목적
- 현재 동영상 강의와 검색을 통해 스프링 부트의 기능과 다른 것들을 배우고 있는 중에 있습니다. 하지만 가만히 보고 있는 것보다 직접 제가 손으로 작성하여 웹 사이트를 만들어보고 싶었습니다. 그리고 제가 배운 기술들을 적용해 보아서 실제 웹 사이트처럼 기능하고, 보여질 수 있게 하고싶습니다.
